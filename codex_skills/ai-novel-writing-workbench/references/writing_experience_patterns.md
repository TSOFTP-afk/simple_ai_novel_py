# Writing Experience Patterns

## Sourcebook Skill

Compress book-level context into concise generation constraints: premise, active characters, world rules, current volume direction, and forbidden contradictions.

## Scene Beats Skill

Before drafting, split a chapter goal into beats: scene entry, character action, information release, conflict escalation, and hook.

## Prompt Builder Skill

Expose the prompt as an author-reviewable draft. Include book, chapter, outline, summary, active Skills, forbidden behavior, and output format.

## Continuity Guard Skill

Check motivation, timeline, setting consistency, foreshadowing, repeated information, and unresolved hooks. Return actionable fixes, not automatic rewrites.

## Style Card Skill

Represent style as abstract constraints: point of view, sentence rhythm, description density, dialogue ratio, emotional temperature, and banned habits.
