关于「灵枢」

灵枢，取自中医经络体系中的核心枢纽——灵台与天枢的合称，寓意此
书写作工具如经络般贯穿长篇创作的每一个关键节点：从世界观搭建、
人物关系编制、章节大纲规划，到正文生成、一致性审计、伏笔追踪
——让 AI 成为作者手上的第二套神经系统，感知、记忆、提醒、补全，
而创作的方向始终由作者掌握。

============================================================

核心架构

灵枢版本对 AI 长篇写作的底层基础设施进行了全面重构，整合了五大
开源项目的技术思路：

一、真相文件系统（Truth）

7 份结构化 Markdown 文件实时追踪全书状态，作为全部 AI 操作的
唯一真相源：

  current_state.md       作品整体状态：大纲、人物、章节进度
  character_matrix.md    人物关系矩阵
  chapter_summaries.md   全部章节摘要
  particle_ledger.md     关键道具 / 世界观条目账本
  pending_hooks.md       伏笔追踪（支持 pending / resolved 状态）
  subplot_board.md       支线看板（自动检测关键词线索）
  emotional_arcs.md      人物情感弧光

每次章节保存后自动同步增量更新，AI 审计时自动交叉验证。

二、RAG 向量检索

  - Qdrant 嵌入式：本地 JSON 文件存储，无需额外服务器
  - 混合检索：向量语义搜索 × BM25 关键词搜索，RRF 融合排序
  - Jina Reranker：可选的二次重排序，提升检索精度
  - 时效衰减：较新章节获得更高权重
  - FallbackEmbedder：内置 TF-IDF 降级方案，零额外依赖

AI 生成的上下文从"固定拼接"升级为"智能检索"——自动找到与
当前章节最相关的前文内容。

三、多 Agent 编排

从单 Prompt 模拟升级为 4 个 Agent 独立调用的 LangGraph 状态图：

  Planner   大纲拆解（3-5 场景计划）    温度 0.3    4096 tokens
  Writer    正文写手（完整叙事）        温度 0.7    8192 tokens
  Auditor   一致性审计（8 维度）        温度 0.25   4096 tokens
  Reviser   修订润色（修复问题）        温度 0.4    8192 tokens

支持取消信号注入，每个 Agent 独立调用 API，结果通过 AgentState
传递。

四、多维度审计引擎

  12 审计维度：情节结构、人物连续性、世界观一致性、逻辑连贯、
    风格稳定、OOC 检测、AI 痕迹检测、伏笔追踪、情感弧光、
    节奏控制、对话质量、模板对齐

  24 条内置规则：加权评分引擎，按维度分组

  OutputValidator：Pydantic 风格输出校验，自动修复缺失字段

  DSLParser：@角色.张三 / @世界.魔法规则 表达式解析，
    将设定条目注入 AI 上下文

五、多模型路由

  budget   经济配置——全部使用 deepseek-chat
  quality  质量优先——Writer + Auditor 使用 deepseek-reasoner
  自定义   set_override() 支持任意模型 / 温度 / Token 配置

界面与交互升级

三级适配卡片：

  书籍级  概览卡片显示 +「AI 总结全书」按钮
          生成卡片显示章节大纲编辑器（默认）

  卷级    概览卡片隐藏
          生成卡片改名为「卷大纲」，显示卷级大纲编辑器 + 保存按钮

  章节级  概览卡片隐藏
          生成卡片显示章节生成大纲

AI 纠正对话：

  嵌入式纠错面板——在多智能体输出后可直接打开对话窗口，逐段
  指出问题，由 AI 修正。

其他优化：

  - 恢复应用图标（exe + 窗口标题栏）
  - 删除 8 个非 AI 自动提炼方法（未配置 API 时直接报错，不再产出假数据）
  - 移除生成卡片左右分栏对照，精简为单编辑器
  - 生成采用章节级大纲，不再混入书籍全局描述

安装与运行

  1. 下载 SimpleAINovelApp-Beta.3.7.exe
  2. 双击运行（首次启动会自动解压运行时）
  3. 在「对话设置」中配置 AI API Key（支持 OpenAI 兼容接口）
  4. 创建书籍 → 创建卷 → 创建章节 → 开始写作

系统要求：Windows 10 / 11 64 位，无需安装 Python 或任何依赖。
